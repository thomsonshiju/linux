LINUX COMMANDS

"Simplify Linux User and Group Management: Learn Commands with Descriptions and Examples"

History

Linux is an open-source, Unix-like operating system that was created by Linus Torvalds in 1991. It is known for being free, flexible, and secure, making it a popular choice for servers, desktops, and embedded systems. Unlike proprietary systems like Windows or macOS, Linux's source code is publicly available, allowing users to modify and customize it to suit their needs. Linux supports a wide range of software applications and can be used in various environments, from personal computing to enterprise-level systems. It also benefits from a large, active community that contributes to its development and maintenance.

init

Purpose: Switch the system to a specified runlevel for starting or stopping services.

Description: Initializes the system to a specific runlevel, determining which services are started or stopped.

Example: Switch to runlevel 3 (multi-user, text mode).

$ init 3

useradd

Purpose: Add a new user to the system and configure their environment.

Description: Adds a new user to the system. By default, the command creates a home directory and assigns default shell settings.

Example: Add a user named "newuser" with a home directory and Bash shell:

$ useradd -m -s /bin/bash newuser

usermod

Purpose: Modify the properties of an existing user account.

Description: Modifies user account settings, such as username or group membership.

Example: Rename "oldname" to "newname":

$ usermod -l newname oldname

userdel

Purpose: Remove a user and optionally their home directory from the system.

Description: Removes a user from the system.

Example: Delete the user and their home directory:

$ userdel -r username

groupadd

Purpose: Create a new group for organizing users.

Description: Creates a new group on the system.

Example: Create a group named "developers":

$ groupadd developers

groupmod

Purpose: Modify an existing group (e.g., rename or change group properties).

Description: Modifies an existing group.

Example: Rename group "developers" to "devs":

$ groupmod -n devs developers

groupdel

Purpose: Remove a group from the system.

Description: Deletes a group from the system.

Example: Remove the group "developers":

$ groupdel developers

passwd

Purpose: Change a user's password.

Description: Changes the password for a user account.

Example: Change the password for "username":

$ passwd username

sudo

Purpose: Execute commands as a superuser or another user with elevated privileges.

Description: Allows a user to execute commands as a superuser or another user.

Example: Update system packages:

$ sudo apt update
